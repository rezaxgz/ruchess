# Ruchess

A rust chess engine, UCI compliant.

## Search

-   Iterative Deepening
-   Transposition Table
-   Move Ordering
-   MVV-LVA
-   Quiescence Search
-   Search Extension

## Evaluation

-   Material
-   Piece Square table
-   Mop Up Evaluation
-   Rook Evaluation
-   Pawn Structure

## Move Generation

The move generation is handled by [jordanbray's chess library](https://github.com/jordanbray/chess)
